Docs - API Reference

The CXP Application Programming Interface (API) follows, RESTful, Hypermedia API standards to facilitate the distribution of digital content (audio, video, images and text) from multiple USG Foreign Affairs agencies to audiences worldwide.

/api/search

Search contents

Parameters
Param Description type
details Show all fields including embedded bool
page Page number integer
limit Number of results to return integer
until Filter by publish date before given date date (YYYY-MM-DD)
since Filter by publish date since given date date (YYYY-MM-DD)
q Search for a string string
type Filter by resource types (article, audio, image, photo_gallery, transcript, video) string
countries Filter by country, given a comma-seperated list of two character country codes string
languages Filter by language given a comma-seperated list of two character language codes string
network Filter by Network, given a lowercase network name string
organization Filter by Organization, given a lowercase org name string
